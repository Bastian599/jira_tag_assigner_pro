# Jira Stichwort-Zuordnung — PRO v3 (FULL)

Enthält **alle** Tabs (Übersicht, P-Labels, Worklog, CSV, Reports) plus die neuen Features:
- 🗓️ **Timesheet (Wochenansicht)** inkl. Wochensumme, CSV-Export & Quick-Add
- 🔐 **Keyring**-Speicherung für Secrets
- 🩺 **Health-Check+**
- Multi-/Single-Projekt, „Nur Lead“-Toggle (Standard **aus**) + Fallback

## Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
